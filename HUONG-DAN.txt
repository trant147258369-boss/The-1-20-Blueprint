CÁC BƯỚC (Windows + Cursor)
  1. Giải nén zip vào folder MỚI, vd:
        C:\Users\ADMIN\Course\the-120-blueprint
     (đừng giải nén đè vào crypto-course)

  2. Mở folder đó trong Cursor.

  3. Cài deps (1 lần, cần internet):
        npm install

  4. Chạy thử local:
        npm run dev
     -> mở http://localhost:3000 xem Hero.

  5. BUILD VERIFY trước khi push (BẮT BUỘC):
        npm run build
     -> phải thấy "✓ Compiled successfully".

  6. (Tùy chọn) đưa lên GitHub repo MỚI + Vercel project MỚI:
        git init
        git add .
        git commit -m "init: The 1:20 Blueprint - Hero (Midnight Jade)"
        git branch -M main
        git remote add origin <URL repo MỚI>
        git push -u origin main
     -> import repo vào Vercel = project mới, domain mới.
     (KHÔNG push vào repo crypto-course cũ.)
